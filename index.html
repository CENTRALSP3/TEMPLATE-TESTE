Scripts

Gerenciamento Completo

Scripts prontos para iniciar, parar, monitorar e reiniciar o servidor Remote Control.

🚀

start.sh — Iniciar

Inicia o servidor com nome personalizado, logs automáticos e suporte a reconexão em caso de falha.

⏹️

stop.sh — Parar

Para o servidor gracefulmente com SIGTERM, fallback para SIGKILL se necessário.

📊

status.sh — Status

Verifica PID, processo, últimas linhas de log e conexões de rede ativas.

🔄

restart.sh — Reiniciar

Reinício rápido: para o servidor e inicia novamente em segundos.

🔔

monitor.sh — Monitorar

Modo daemon com notificações desktop e webhook (Slack/Discord) e auto-restart opcional.

⚙️

Serviço 24/7

Inclui unit systemd (Linux) e script PowerShell para Agendador de Tarefas (Windows).
